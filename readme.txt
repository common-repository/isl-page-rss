== Description ==

Simple plugin that adds all static pages to your Wordpress Blog RSS feed.

== Installation ==

1. Upload the `isl-page-rss` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Your pages will now be displayed in your RSS feed.
